# Repo for learning about AI and machine learning from scratch

This repository is dedicated to learning about artificial intelligence (AI) and machine learning (ML) from scratch. It contains programming exercises and code for learning the fundamentals of AI and ML, including algorithms, data structures, and practical applications. The goal is to provide a hands-on approach to understanding these concepts through coding and experimentation. Whether you're a beginner or looking to deepen your knowledge, this repo offers resources and examples to help you on your AI and ML learning journey.


## Getting StartedTo get started with this repository, follow these steps:
1. Clone the repository to your local machine using the command:
   ```
   git clone
    ```
2. Navigate to the cloned directory:
    ```
    cd your-repo-name
    ```
3. Explore the different folders and files to find exercises and code examples related to AI and ML.
4. Follow the instructions in the README files within each folder to understand the exercises and how to run the code.
5. Experiment with the code, modify it, and try to implement your own versions of the algorithms and applications.

## Numpy and Pandas
The repository includes exercises and code examples for learning about Numpy and Pandas, which are essential libraries for data manipulation and analysis in Python. These exercises will help you understand how to work with arrays, data frames, and perform various operations on them.
